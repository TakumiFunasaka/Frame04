ROGUELIKE MECH CARD BATTLE

西暦2187年――人類は「魔導機関」の発見により、新たなエネルギー革命を迎えた。

魔導機関を搭載した機動兵器「フレーム」は軍事の中核となり、

各国は最強の小隊を競う国際競技リーグ「セクターダイブ」を開催。

プレイヤーは4機のフレームで小隊を編成し、

無限に続く深層セクターへ挑む。到達深度がそのままランク。

どこまで潜れるか――それがすべてだ。

// ゲームの流れ

1. 小隊編成 ── 10種のフレーム（メカ）から4機を選ぶ

2. 出撃 ── 全戦エリート級。1 Act = 10戦（弱→中→強→ボス）

3. 戦闘 ── ターン制カードバトル。ENを管理してカードを使う

4. 報酬 ── 勝利でカード発見、レリック獲得

5. 無限Act ── Actは無限に続く。敵はAct毎にスケーリング。全滅するまで戦え

// デッキ

4機 × 3枚 = 初期12枚のデッキで開始

戦闘報酬で新カードを発見してデッキに追加できる

イベントでカードを除去してデッキを圧縮することも可能

// レリック

エリート・ボス撃破でレリックを獲得。パッシブ効果が永続発動

攻撃強化 / 防御強化 / 状態異常強化 / 経済系など多種多様

// 大破（デッドドロー）

HPが0になった機体のカードはすべて使用不可に。手札を圧迫し続ける

メディックの「リブート」で復活可能

// ターンの流れ

1. ターン開始: EN+3（上限5、繰り越しあり）。山札から5枚ドロー

2. プレイヤーフェーズ: ENが許す限りカードをプレイ

3. END TURN: 残り手札→捨て札。敵が行動。過熱ダメージ処理

4. 山札0枚で捨て札をシャッフルして山に戻す

// EN（エネルギー）

4機共有のリソース。毎ターン+3、最大5までストック

各カードにENコストがあり、コスト分を消費してプレイ

// インテント

敵は「次に誰に何をするか」を事前表示する

挑発カードで敵のターゲットを自機に強制変更できる

// レリック

パッシブ効果を持つアーティファクト。所持数に上限なし

エリート・ボス撃破、イベント等で獲得

// 3ステータス

OUT（出力）攻撃ダメージ倍率: baseDmg x (1 + OUT x 0.04)
SHL（外殻）最大HP倍率: baseHP x (1 + SHL x 0.03)
フィールド倍率: base x (1 + SHL x 0.04)
CTRL（制御）回復倍率: baseHeal x (1 + CTRL x 0.04)
過熱/脆弱の付与量もスケーリング

// ステータスについて

各フレームにはbaseStats（固定値）があり、戦闘中のダメージ計算に使用される

// 状態異常（敵に付与）

過熱(N)ターン終了時にNダメージ。毎ターン-1減衰。スタック可
CTRLスケーリングあり
脆弱(N)次にダメージを受けた時+N。被弾1回で消費
CTRLスケーリングあり
感電(T)Tターンの間、受けるダメージx1.2倍
スケーリングなし

// 速度軸

加速(+N%)味方に付与。被弾率が100-N%に低下
減速(-N%)敵に付与。命中率が100-N%に低下

5ターン持続。重ねがけは加算。相殺も加算

// フィールド（味方バフ）

ダメージ吸収バリア。毎ターン開始時にリセットされる（使い切り）

SHLスケーリングあり

// 編成のコツ

アタッカー2 + タンク/サポート1 + ヒーラー/デバッファ1 がバランス型

アタッカー4の速攻編成もあり。大破リスクは高いが殲滅力が段違い

// カード発見

初期デッキは各フレーム3枚ずつ計12枚。戦闘報酬でカードを発見してデッキを強化

イベント「データ整理」でカードを除去してデッキを圧縮することも重要

// レリック

エリート・ボスを倒すとレリックを獲得できる

攻撃系 / 防御系 / 状態異常系 / 速度系 / 経済系など効果は多彩

ボスレリックは特に強力。ビルドの方向性を決める要

// 戦闘間イベント

戦闘後40%の確率でランダムイベントが発生:

整備=HP30%回復 / 最適化=カード+化 / データ整理=カード除去

残骸発見=レリック獲得 / 賭け=HP消費でレリック獲得チャンス

// 重要メカニクス

敵のインテントを見て、挑発でターゲットをタンクに集める

フィールドは毎ターンリセット → 「今ターン守る」判断が重要

大破は致命的 → メディックがいればリブートで復活可能
