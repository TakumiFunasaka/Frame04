ROGUELIKE MECH CARD BATTLE

西暦2187年――人類は「魔導機関」の発見により、新たなエネルギー革命を迎えた。

魔導機関を搭載した機動兵器「フレーム」は軍事の中核となり、

各国は最強の小隊を競う国際競技リーグ「セクターダイブ」を開催。

プレイヤーは4機のフレームで小隊を編成し、

無限に続く深層セクターへ挑む。到達深度がそのままランク。

どこまで潜れるか――それがすべてだ。

// ゲームの流れ

1. 小隊編成 ── 10種のフレーム（メカ）から4機を選び、チューニングポイント(TP)を配分

2. 出撃 ── 全戦エリート級。1 Act = 10戦（弱→中→強→ボス）

3. 戦闘 ── ターン制カードバトル。ENを管理してカードを使う

4. 報酬 ── 勝利でTP獲得 → ステ強化、カード最適化(+化)、アイテム入手

5. 無限Act ── Actは無限に続く。敵はAct毎にスケーリング。全滅するまで戦え

// デッキ

4機 × 5枚 = 計20枚の固定デッキ（追加・削除なし）

カードは戦闘報酬で最適化(+)して永続強化できる

// 大破（デッドドロー）

HPが0になった機体のカード5枚はすべて使用不可に。手札を圧迫し続ける

メディックの「リブート」で復活可能

// ターンの流れ

1. ターン開始: EN+3（上限5、繰り越しあり）。山札から5枚ドロー

2. プレイヤーフェーズ: ENが許す限りカードをプレイ

3. END TURN: 残り手札→捨て札。敵が行動。過熱ダメージ処理

4. 山札0枚で捨て札をシャッフルして山に戻す

// EN（エネルギー）

4機共有のリソース。毎ターン+3、最大5までストック

各カードにENコストがあり、コスト分を消費してプレイ

// インテント

敵は「次に誰に何をするか」を事前表示する

挑発カードで敵のターゲットを自機に強制変更できる

// アイテム

最大3個所持。戦闘中にクリックで即時使用（使い切り）

HPポーション / フィールドパック / ENセル の3種

// 3ステータス

OUT（出力）攻撃ダメージ倍率: baseDmg × (1 + OUT × 0.04)
SHL（外殻）最大HP倍率: baseHP × (1 + SHL × 0.03)
フィールド倍率: base × (1 + SHL × 0.04)
CTRL（制御）回復倍率: baseHeal × (1 + CTRL × 0.04)
過熱/脆弱の付与量もスケーリング

// TP（チューニングポイント）

初期24ptを4機に自由配分。戦闘勝利で追加TP獲得

各フレームにはbaseStats（固定10pt分）があり、TPはその上に加算される

// 状態異常（敵に付与）

過熱(N)ターン終了時にNダメージ。毎ターン-1減衰。スタック可
CTRLスケーリングあり
脆弱(N)次にダメージを受けた時+N。被弾1回で消費
CTRLスケーリングあり
感電(T)Tターンの間、受けるダメージ×1.2倍
スケーリングなし

// 速度軸

加速(+N%)味方に付与。被弾率が100-N%に低下
減速(-N%)敵に付与。命中率が100-N%に低下

5ターン持続。重ねがけは加算。相殺も加算

// フィールド（味方バフ）

ダメージ吸収バリア。毎ターン開始時にリセットされる（使い切り）

SHLスケーリングあり

// 編成のコツ

アタッカー2 + タンク/サポート1 + ヒーラー/デバッファ1 がバランス型

アタッカー4の速攻編成もあり。大破リスクは高いが殲滅力が段違い

// TP配分

「おまかせ配分」が便利。迷ったらまずそれで試そう

アタッカーにOUT全振り、タンクにSHL全振りが基本

// 戦闘間イベント

戦闘後40%の確率でランダムイベントが発生:

整備=HP30%回復 / 最適化=カード+化 / 補給=アイテム

賭け=HP消費でTP獲得チャンス / 偵察=次の敵情報

// 重要メカニクス

敵のインテントを見て、挑発でターゲットをタンクに集める

フィールドは毎ターンリセット → 「今ターン守る」判断が重要

大破は致命的 → メディックがいればリブートで復活可能
