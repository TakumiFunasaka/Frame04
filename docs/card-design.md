# FRAME:04 カードデザイン

> 共通メカニクス（状態異常3種+速度軸/フィールド）の定義は `game-design.md` セクション6を参照。
> 全カードの正式データは `prototype/frames.json` を参照。

## 基本ルール
- 各フレーム8枚（初期3枚 + 発見5枚）
- 初期デッキ: 3枚 × 4機 = 12枚。戦闘報酬でカード追加して成長
- EN（エネルギー）: ターン開始時+3（繰り越しあり、上限5）
- 毎ターン5枚ドロー
- カードコスト: 0〜3 EN

### カード最適化（+化）ルール
イベント「最適化」で獲得。ランダムなカード1枚を永続強化。

| 効果カテゴリ | +効果 | 例 |
|---|---|---|
| ダメージ値 | +2 | パイルバンカー 13→15 |
| フィールド値 | +2 | フルガード 5→7 |
| 回復値 | +2 | リペアナノ 6→8 |
| 状態異常スタック(過熱/脆弱) | +1 | 過熱(3)→過熱(4) |
| 状態異常ターン(感電) | +1T | 感電(1T)→感電(2T) |
| 自傷ダメージ | -2（軽減） | 自機HP-8→自機HP-6 |
| 条件付きボーナス | +2 | バックスタブの+5→+7 |

## シナジー設計

### フレーム間シナジー例
- **脆弱コンボ**: ガンナー/クラッカーが脆弱付与 → ストライカーのフィニッシュブローで消費＝大ダメージ
- **過熱持久**: ブラスター+ジャマーが過熱を重ねがけ → シールダーで耐えながら燃やし尽くす
- **感電バースト**: オーバーロード+ジャマーが感電付与 → 感電中(×1.2倍)に全機で集中砲火
- **速度制御**: ファントムが敵に減速 → 敵の攻撃がミス。ブースター/ファントムが味方に加速 → 二重防御
- **過熱+感電**: ジャマーのシステムクラッシュ + オーバーロードのチェインショック → 過熱状態の敵に感電で追加ダメージ
